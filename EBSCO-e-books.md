[EBSCO e-books](https://www.ebsco.com/products/ebooks) is a ebook library service of EBSCO. Technical specification for (meta)data upload are provided [here](https://www.ebsco.com/publisher-support/ebooks).

* File transfer: [[FTP]]
* Content files: [[EPUB]], [[PDF]]
* Metadata format: [[ONIX 2.1]], proprietary [[XLSX]] file
